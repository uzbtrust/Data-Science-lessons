# Data-Science-lessons

## O'quv Materiallari

Bu repository Data Science asoslarini o'rganish uchun tuzilgan kurs materiallari hisoblanadi.

### Kurs Tuzilishi:
- **NumPy**: Array va matematika amaliyotlar
- **Pandas**: Data analiz va manipulyasiya
- **Data Warehouse**: Ma'lumot saqlash tizimlari
- **Data Preparation**: Ma'lumotlarni tayyorlash
- **Harmonization**: Ma'lumotlarni uyushlash
- **Visualization**: Grafikal tahlil
- **ML**: Mashinali o'rganish asos tushunchasi
- **Regression**: Regressions modellar
- **Classification**: Klassifikatsiya masalasi
- **Clustering**: Klasterizatsiya
- **Deep Learning**: Chuqur o'rganish

Barcha materiallari ipynb formatida va amaliyot vazifalarini o'z ichiga oladi.
